## Használati útmutató

Használat:

1. Először állítsuk be az időt, a képernyő jobb alsó sarkában található beviteli mező segítségével.
2. Ezután indítsuk el a missziót, a beviteli mező melletti gomb használatával.

A futás során:
- az aktuális adatok folyamatosan láthatók a jobb felső sarokban,
- a bal oldalon látható térképpel folyamatosan nyomon követhetjük a rover-t,
- a rover döntéseit láthatjuk az eseménynaplón is, amely a rover aktuális adatai alatt látható.
